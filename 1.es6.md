es6 必备知识

1. 变量

```js
var // 重复声明，函数级
let // 不能重复声明，块级，变量
const // 不能重复声明，块级，常量
```

2. 解构赋值（声明和赋值都放到一起）

```js
//解构 表示等号左边和右边结构类似
//数组必须位置相同
//对象解构时必须key值相同
let { length } = [1, 2, 3]; //可以获取数组的长度
//对象key值为关键字需要取别名 ：xx
//如果想设置某个属性的默认值，采用=号的方式赋值
```

3. 模版字符串

```js
//模板字符串取代了原油的字符串拼接功能
//可以支持换行可以支持取值 ${}
includes()      //是否包含
startsWith()    //是否以xx开头
endsWidth()     //是否以xx结尾
padStart(a,b) padEnd(a,b) //补全a为位数，b为补的值
//面试题：如何实现一个类模板字符串的功能
let name = "sjs";
let age = 30;
let str = "hello~${name}今年${age}了！";
str = str.replace(/\$\{([^}]*)\}/g, function () {
  console.log(arguments);
  return eval(arguments[1]);
});

```

```js
//1、没有function这个关键字

//4、如果没有return可以省略大括号
//5、没有this指向
```
